AI Email Scam Detector

Live Project: mailscammers.loveable.app

Developed an AI-powered web application that detects phishing and scam emails using Google Gemini AI and Google APIs. The platform analyzes suspicious email content and provides scam detection results through a clean and responsive user interface. Integrated AI prompt processing, API handling, and real-time analysis features while deploying the application as a live production-ready website.

Key Features:
AI-based scam and phishing email detection
Google Gemini API integration
Responsive React frontend UI
Real-time email analysis system
Live deployed web application
Tech Stack:

React.js, JavaScript, Vite, Gemini API, Google APIs, HTML, CSS
